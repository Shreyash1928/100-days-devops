# 🚀 100 Days of DevOps

Welcome to my **100 Days of DevOps** learning journey!

This repository documents my day-by-day progress as I build practical skills in Linux, Docker, Kubernetes, Ansible, Terraform, AWS, CI/CD, and other DevOps tools through hands-on practice.

## 🎯 Goal

The objective of this challenge is to:

- Build a strong DevOps foundation.
- Learn by doing hands-on exercises every day.
- Maintain consistency through daily practice.
- Create a portfolio showcasing real-world DevOps skills.
- Document commands, notes, and projects for future reference.
